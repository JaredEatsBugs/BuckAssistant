CSE 2231 - Software II: Software Development and Design
Description
Data representation using hashing, search trees, and linked data structures; algorithms for sorting; using trees for language processing; component interface design; best practices in Java. Prereq: 2221. Concur: 2321. Not open to students with credit for 2231.01, 321, or 421. Enrollment Requirements: Prereq: 2221. Not open to students with credit for 2231.01, 321, or 421.
